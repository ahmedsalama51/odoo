Account Bank Statement Auto Reconciliations

New features, new helper fields

This module add several functions of the original check module.
- Show account field on bank statement line to do automatic reconcile.
- Add sequence field on each account bank statement.
- Add constrain on statements to add sequence only 1 time for each statement.
- Add Discharge field on each account bank statement to show if it is [Pay To/Income From].
- Add Receivable Name field on each account bank statement.
- Add Actual Exchange Rate field on each account bank statement.
- Simplification of management in general.
